HTML Basics with Tag Explanations

HTML Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <!-- Your content goes here -->
    <h1>Hello World!</h1>
    <p>This is a paragraph.</p>
</body>
</html>

HTML Document Structure Tags

<!DOCTYPE html> - Declares the document type and HTML version (HTML5 in this case).

<html> - The root element of an HTML page. The lang attribute specifies the language of the document.

<head> - Contains meta information about the document, links to stylesheets, scripts, and the title.

<meta> - Provides metadata about the HTML document (character encoding, viewport settings, etc.).

<title> - Sets the title of the document (shown in browser tab/window title).

<body> - Contains the visible page content.

<h1> - Main heading (there are also h2-h6 for subheadings).

<p> - Defines a paragraph of text.

Common HTML Tags with Explanations

<!-- Links -->
<a href="https://example.com" target="_blank">Visit Example</a>

<!-- Images -->
<img src="image.jpg" alt="Description" width="200">

<!-- Lists -->
<ul>
    <li>Unordered list item</li>
</ul>

<ol>
    <li>Ordered list item</li>
</ol>

<!-- Divisions and Sections -->
<div class="container">Content</div>
<section id="main">Section content</section>

Common HTML Tags Explained

<a> - Creates hyperlinks. The href attribute specifies the URL, and target="_blank" opens the link in a new tab.

<img> - Embeds images. src specifies the image source, alt provides alternative text, and width sets the width in pixels.

<ul> - Unordered (bulleted) list. Contains <li> list items.

<ol> - Ordered (numbered) list. Also contains <li> items.

<div> - Generic container for flow content. Used with CSS for layout.

<section> - Semantic section of content, typically with a heading.

CSS Fundamentals with Property Explanations

Basic CSS Syntax

/* Selector targets all <p> elements */
p {
    color: blue;
    font-size: 16px;
    margin: 10px 0;
}

/* Class selector */
.highlight {
    background-color: yellow;
    padding: 5px 10px;
    border-radius: 4px;
}

CSS Properties Explained

color - Sets the text color (can be named colors, hex, rgb, etc.).

font-size - Sets the size of the font (px, em, rem, %, etc.).

margin - Sets the outer spacing of an element. Can have 1-4 values (top, right, bottom, left).

background-color - Sets the background color of an element.

padding - Sets the inner spacing of an element (between content and border).

border-radius - Rounds the corners of an element's border.

Box Model Explained

.box {
    width: 300px;
    height: 200px;
    padding: 20px;
    border: 2px solid black;
    margin: 30px;
    box-sizing: border-box;
}

Box Model Properties

width / height - Sets the dimensions of the content area.

padding - Space between content and border. Adds to total element size unless box-sizing is border-box.

border - The line around the element. Syntax: width style color.

margin - Space outside the border, separating elements.

box-sizing: border-box - Makes width/height include padding and border (more intuitive sizing).

Flexbox Layout

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    gap: 20px;
    flex-wrap: wrap;
}

.item {
    flex: 1;
    min-width: 200px;
}

Flexbox Properties

display: flex - Enables flexbox layout for direct children.

justify-content - Aligns items along the main axis (horizontal by default).

align-items - Aligns items along the cross axis (vertical by default).

gap - Sets space between flex items.

flex-wrap - Allows items to wrap to new lines if needed.

flex - Shorthand for flex-grow, flex-shrink, and flex-basis.

JavaScript Basics with Explanations

Variables and Data Types

// Variables
let message = "Hello, World!"; // String
const PI = 3.14; // Constant number
var oldVariable = 10; // Old way (avoid)

// Data Types
let isActive = true; // Boolean
let person = { // Object
    name: "John",
    age: 30
};
let colors = ["red", "green", "blue"]; // Array

JavaScript Variables and Data Types

let - Declares a block-scoped variable that can be reassigned.

const - Declares a block-scoped constant that cannot be reassigned.

var - Old way to declare variables (function-scoped, avoid in modern code).

Primitive Types - String (text), Number (integer or decimal), Boolean (true/false), Null, Undefined.

Object - Collection of key-value pairs (properties and methods).

Array - Ordered list of values (can mix types).

Functions Explained

// Function declaration
function greet(name) {
    return `Hello, ${name}!`;
}

// Arrow function (ES6+)
const add = (a, b) => {
    return a + b;
};

// Immediately Invoked Function Expression (IIFE)
(function() {
    console.log('This runs immediately');
})();

JavaScript Functions

function - Traditional function declaration (hoisted, can be called before declaration).

arrow function - Shorter syntax, doesn't have its own this context.

parameters - Variables listed in the function definition (name in the example).

arguments - Values passed to the function when called.

return - Specifies the value to return from the function.

IIFE - Immediately Invoked Function Expression, runs as soon as defined.

Python Basics with Explanations

Variables and Data Types

# Variables
message = "Hello, World!"  # String
pi = 3.14159  # Float
count = 10  # Integer
is_active = True  # Boolean

# Collections
colors = ["red", "green", "blue"]  # List (mutable)
person = {"name": "Alice", "age": 25}  # Dictionary (key-value pairs)

Python Variables and Data Types

Variables - No declaration needed, created when first assigned.

Strings - Text data, enclosed in single or double quotes.

Numbers - Integers (whole numbers) and floats (decimals).

Boolean - True or False values (must be capitalized).

List - Ordered, mutable collection of items.

Dictionary - Unordered collection of key-value pairs.

Functions and Control Flow

# Function definition
def greet(name):
    return f"Hello, {name}!"

# If-elif-else statement
if age >= 18:
    print("Adult")
elif age >= 13:
    print("Teen")
else:
    print("Child")

# For loop
for color in colors:
    print(color)

Python Functions and Control Flow

def - Keyword to define a function.

return - Specifies the value to return from the function.

f-strings - Formatted string literals (f before quotes allows {variables}).

if-elif-else - Conditional statements for decision making.

for loop - Iterates over items in a sequence (list, string, etc.).

indentation - Python uses indentation (4 spaces) to define code blocks.
